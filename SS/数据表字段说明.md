# 数据表字段说明

> 本文档基于 `backend/app/models/` 下的实际模型定义编写，与代码保持一致。
> 向量维度统一为 **1024**（BAAI/bge-m3 模型）。标签系统已从旧的 jsonb tags 字段迁移到独立的 TagDefinition + ContentTag 关联表。

## 修改指引

如果你需要：

- **增加/修改前端展示列** -- 修改对应前端页面组件（`frontend/src/pages/Documents.tsx`、`frontend/src/pages/Communications.tsx` 等）
- **增加数据库字段** -- 修改 `backend/app/models/` 下的模型文件，然后运行 `cd backend && alembic revision --autogenerate -m "描述"` 创建迁移
- **增加 API 返回字段** -- 修改 `backend/app/schemas/` 下的 Pydantic 模型
- **调整 ETL 映射规则** -- 修改 `backend/app/services/etl/` 下的 pipeline 文件（transformer.py、enricher.py 等）
- **管理标签** -- 标签定义通过 `/api/tags` 接口管理，内容打标通过 `/api/content-tags` 接口或 ETL 自动打标
- **配置提取/清洗规则** -- 通过前端数据导入页（`/data-import`）或 `/api/extraction-rules`、`/api/cleaning-rules` 接口管理
